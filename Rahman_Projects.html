Projects

Project: Scratch Project

Course: CSE

Duration:3-4 weeks

Type: Pair

Objective: Create an interactive scratch game that can be played by other classmates.

For the scratch project, I began by brainstorming different ideas and looking up how I could recreate these visions in Scratch. I eventually chose to use a space battleship-type game and began to look up images to use for sprites. Once all of the base code was done, I tested my code constantly until the game neared protection. To end the project, we had a class gallery-walk where everyone got to try eachother's game. It was definitely a great experience.

Project: App Inventor Project

Course: CSE

Duration: 3-4 weeks

Type: Pairs

For the App Inventor Project, we began by forming pairs and making a team desicion on which game template we wpuld use. Once we chose the mini golf base, we completed the base code and confirmed that there were no bugs by testing through the tablets. After getting checked off, we changed certain aspects of the game, adding new levels, difficulties, and changing the sprites to become more interactive.This allowed the user to play for a longer period of time because of all the different levels.
